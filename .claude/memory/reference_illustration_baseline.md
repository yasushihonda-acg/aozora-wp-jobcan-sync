## メインキャラクター仕様 (顔造形 — Image #21 close-up 詳細観察)

### 全体
- 日本人女性、**30 代前半 (mature, composed)**、20 代に見えると違う
- 柔らかい卵型の輪郭、顎は細く優雅にすぼまる (gracefully tapered jaw)
- 頬骨にうっすら定義 (subtle cheekbone)、丸顔ではない
- 印象: **知的・落ち着き・控えめな大人っぽさ + 親しみやすさ** (無邪気・幼さ NG)

### 眼 (最重要識別要素) — Image #22 厳密観察
- **細長めのアーモンド型** (丸すぎない、大きすぎない、若々しすぎない)
- 瞳: 明るい茶色 (warm brown)、瞳孔は黒、繊細なハイライト 1 点
- **目尻が顕著に下がる** (strongly downturned outer corners、優しい大人の印象の最大要因)
- 二重幅は浅い〜奥二重 (くっきりした二重ではない)
- 上まぶたに繊細なまつ毛 (細い短い線で表現、束感はなし)
- 白目部分は清く、虹彩の輪郭線はくっきり描かれる
- 視線: viewer 向きの正面、温かく落ち着いた目線

### 眉
- 細く短く整っている (細眉)
- 控えめな緩いアーチ、眉尻は自然に下がる
- 眉色は黒、線で軽く表現

### 鼻
- 高さ控えめ、線で軽く示唆 (鼻筋を強調しない)
- 鼻先は小さく丸い
- 鼻翼に淡い影

### 口
- 控えめな微笑、**上歯がほんの少し覗く程度** (歯列を見せない、下歯は絶対に見せない、grin は絶対 NG)
- 上唇は薄め、下唇はふっくら
- リップ: 自然なピンク〜サーモン (warm coral pink)、控えめ
- 口角は左右やや上がる、左右対称
- ※「無邪気な子供の笑顔」ではなく「落ち着いた大人の微笑」

### 頬・肌
- 暖色 (warm peach)、清潔感、柔らかい質感
- 頬に淡いピンクの紅潮 (僅か、ぼかし)
- シェーディング最小、フラット気味

### 眼鏡 (キャラ識別要素)
- **べっ甲フレーム** (tortoiseshell、茶色 + 黒の斑模様)
- **丸めの太縁** (medium-thick rim)、レンズはラウンド気味のオーバル
- ブリッジは細め、テンプル (つる) も細め
- 鼻パッド可視

### 髪
- 黒髪 (純黒〜墨色)
- **低めの位置でゆるくまとめたバン (お団子)**、後頭部下部に小さくまとめる
- 前髪: 分け目から左右にふんわり、額が少し見える程度
- **顔まわりに繊細な後れ毛が複数本** (耳の前 2-3 本、額の左右、こめかみ付近)
- 髪の毛束は細い個別線で描画、塗りつぶしではない

### 耳・装飾
- **小さな金のフープピアス** (右耳側に見えれば OK)

### 表情バリエーション (今後の生成で許容する範囲)
- 微笑 (上歯少し見せ) ← baseline 既定
- 真顔・伏目 (考え込み・記入時)
- やや横向き・斜め (3/4 view)
- 笑顔 (歯を多く見せる) ← 控えめに留める
- 上記すべて、目の形・眉・鼻・口角・後れ毛のパターンは保持必須

### 体格・印象
- 細身でほっそりした体格
- 知的・落ち着き・親しみ
- 介護・医療事業者の「相談しやすいプロ」イメージ

### 服装 (既定: 介護スクラブ)
- コーポレートカラー `#00c4cc` の V ネックスクラブ
- 白パイピング (襟・袖口)
- 首から白 ID ストラップ + クリアケースの ID カード
- 胸ポケット 1 つ

職種派生 (許容):
- 事務シーン: 同色系の襟付きシャツ + カーディガン
- 経営/管理シーン: テーラードジャケット (ターコイズ近似色)
- ※派生時も顔・髪・眼鏡は完全一致

## 画風 (厳密一致)

baseline 観察に基づく画風スペック:
- **繊細な線画** (細く、しなやかで、機械的にならない)
- **透明水彩淡彩** (色は重ねず、淡くフラットに置く)
- **シェーディング最小** (顔・服に明暗を強くつけない)
- **紙質感は控えめ** (パリッとした洋紙風、ザラザラした和紙質感 NG)
- **線・色のにじみ控えめ** (過剰水彩・ぼかし NG)
- **色温度暖色寄り** (warm peach 肌、淡いベージュ背景、淡い緑)
- **アクセント** = ターコイズ `#00c4cc` (スクラブ + ペンダントランプ等)
- **背景の窓・植物** = 淡い緑、自然光のソフトな差し込み
- **NG**: アニメ調の黒目ハイライト過剰、グロッシーな質感、油彩風厚塗り、3D 風シェーディング、過度な水彩にじみ

## How to apply

**MUST**: テキストプロンプトのみの `v1/images/generations` は使用禁止。同一キャラクター再現が原理的に困難 (2026-06-29 実証、4 カテゴリ全 NG)。

**唯一の正解手法**: `v1/images/edits` で baseline 画像 (`generated-images/gpt-image-nurse-20260629-190600.png`) を `image[]` に渡す reference-based generation。

```bash
curl -X POST "https://api.openai.com/v1/images/edits" \
  -H "Authorization: Bearer $KEY" \
  -F "model=gpt-image-2" \
  -F "image[]=@generated-images/gpt-image-nurse-20260629-190600.png" \
  -F "prompt=<シーン記述。キャラクター指定は不要、reference 画像で自動継承>" \
  -F "size=1536x1024" \
  -F "quality=high" \
  -F "output_format=png"
```

プロンプトには:
- シーン (場所・小道具・副役)
- 必要なら "Keep the same character, same outfit, same illustration style as reference."
- 服装派生がある場合のみ outfit 指定 (例: 経営層 → cardigan)

生成後、baseline と顔を見比べて同一人物確認。逸脱があれば再生成。
